## Готовая сборка Laravel 9

Сборка включает в себя настроенное локальное Docker окружение с:

- nginx
- php-fpm
- php-cli
- postgresql

Из сборки удалены все ненужные файлы и лишние пакеты.

Инфраструктурно добавлены:
- Контроллер
- CycleORM
